## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Stack

- **React** — UI
- **TypeScript** — types
- **Vite** — dev server and bundler
- **Tailwind CSS** — styling
- **ESLint** — linting
- **Lucide React** — icons
- **clsx** / **tailwind-merge** — class names
